# Виджет "Генератор XML" для amoCRM

Этот виджет добавляет кнопку "Создать XML" в карточку сделки в amoCRM. При нажатии на кнопку отправляется запрос на сервер для генерации и скачивания XML-файла.

## Структура файлов

- `manifest.json`: Файл конфигурации виджета.
- `script.js`: Основной файл логики виджета.
- `i18n/ru.json`: Файл локализации для русского языка.
- `images/`: Папка для изображений виджета.

## Установка

1.  **Подготовьте файлы:**
    *   При необходимости замените изображения-плейсхолдеры в папке `images` на ваши реальные логотипы.
    *   При необходимости добавьте файлы локализации для других языков в папку `i18n`.

2.  **Загрузите на сервер:**
    *   Загрузите все файлы виджета (`manifest.json`, `script.js`, папки `i18n` и `images`) на ваш веб-сервер.

3.  **Установка в amoCRM:**
    *   В вашем аккаунте amoCRM перейдите в `Настройки` -> `Интеграции`.
    *   Нажмите "Создать интеграцию" -> "Внешняя интеграция".
    *   В поле "Ссылка на manifest.json" укажите **полный публичный URL** до вашего файла `manifest.json`.
    *   Активируйте интеграцию и сохраните.
    *   Перейдите в настройки виджета и укажите ваш **Bearer Token** для доступа к API.

## Использование

1.  Откройте любую сделку в amoCRM.
2.  В правой части экрана появится виджет с кнопкой "Создать XML".
3.  Нажмите на кнопку. Начнется скачивание XML-файла, сгенерированного для данной сделки.
